# 口播稿

你有没有想过，世界在最底层，运行的规则跟日常完全不一样？

一只猫可以同时是死的和活的。

一个粒子能同时穿过两条缝。

两个粒子隔着整个银河系，也能瞬间感应对方。

这不是魔法。这是现代物理学最硬核的基石——量子力学。

---[build]

我们从小就被教，东西分两种。粒子，像子弹，有明确位置，沿确定轨迹飞。波，像水波，弥散开来，会绕过障碍、互相干涉。

但在量子世界，这条分界线彻底糊了。

有一个经典实验，叫双缝干涉。

你拿电子枪，朝开了两条缝的挡板射击。后面是屏幕。

如果电子是粒子，它要么走左缝，要么走右缝。屏幕上应该出现两条亮纹。

但实验结果呢？屏幕上出现了明暗相间的干涉条纹。跟水波穿过双缝一模一样。

更离谱的是，哪怕你让电子一颗一颗发射。每次只过去一个。积累久了，还是干涉条纹。

一个电子，怎么就同时通过了两条缝？

就好像它是一朵概率云，同时走了所有可能的路径，最后自己跟自己干涉了。

波粒二象性告诉我们：量子物体既不是经典的粒子，也不是经典的波。它是什么，取决于你怎么观测它。

---[build]

如果一个电子能同时走两条缝，那它是不是处于「既走左又走右」的状态？

没错。这就是叠加态。

量子系统在被测量之前，可以同时处于多种可能状态的线性叠加。

就像一枚正在旋转的硬币。既不是正面，也不是反面，而是正与反的共存。

物理学家薛定谔为了讽刺这个观点，1935 年提出了一个思想实验。

把一只猫关进密封盒子。里面放一个放射性原子，和一瓶毒气。

原子衰变，机关触发，毒气释放，猫死。不衰变，猫活。

按量子力学，原子处于衰变与未衰变的叠加态。那猫呢？死和活的叠加？

这显然荒谬。我们从没见过半死不活的猫。

---[turn]

但正是这个荒谬的猫，点中了量子力学的核心难题。

微观世界的叠加态，为什么在宏观世界消失了？

答案是：那只猫本身就是测量装置。

原子一旦跟猫、毒气瓶这些宏观物体发生纠缠，叠加态迅速退相干。变成确定的死或活。

所以猫从来不会真的一边死一边活。但微观粒子，确实可以。

一个量子比特，可以同时是 0 和 1。这让量子计算机能并行处理海量可能性，带来指数级算力提升。

---[turn]

还有更颠覆的。

在寻常世界，你看月亮，月亮就在那里，跟你无关。

但在量子世界，观测本身会改变被观测对象。

回到刚才的双缝实验。你在两条缝旁边装上探测器，想看电子到底走了哪条缝。

神奇的事发生了。干涉条纹，立刻消失。

屏幕上只剩两条单缝衍射的叠加。电子乖乖地只走了一条缝。

你的偷窥，迫使电子放弃了波动性，展现出粒子性。

这种测量导致的波函数坍缩，争议至今不断。

哥本哈根诠释说：测量让量子态从多种可能性中随机选一个。多世界诠释说：没有坍缩，每次测量宇宙都分裂成多个分支。

不管哪种，都承认一件事。观测者和量子系统之间，有一种根本的联系。

---[data]

爱因斯坦最受不了的，就是量子纠缠。他叫它「鬼魅般的超距作用」。

两个粒子一旦发生纠缠，不管相隔多远。哪怕一个在地球，一个在火星。

测其中一个的状态，另一个瞬间就知道了，呈现出确定关联。好像它们之间有一根无形的线。

爱因斯坦认为这太荒谬了。但实验反复证实，他错了。

2017 年，中国墨子号卫星，实现了千公里级的量子纠缠分发。

基于纠缠的量子隐形传态，能把粒子量子态传输到远处。不是传送物质，是传送信息。科幻里的超时空传输，有了现实雏形。

---[data]

还有一个更反直觉的——量子隧穿。

经典物理说，小球动能不够，就翻不过山包，半路滚回来。这是铁律。

但在量子世界，粒子有一定概率像幽灵一样，直接穿透能量壁垒，出现在山的另一侧。

这不是魔法。太阳发光就靠这个。

按经典物理，太阳内部温度根本不够克服质子间的静电排斥。但质子通过量子隧穿，穿透了库仑壁垒，聚变才得以发生。

你手机里的闪存，擦写数据靠的是电子隧穿。

---[release]

在量子的王国里，没有绝对的不可能。只有概率的大小。

---[close]

你可能想问，量子力学这么玄，跟我有什么关系？

说真的，没有量子力学，就没有整个信息时代。

半导体物理，完全建立在量子理论之上。能带理论解释了为什么硅是半导体。晶体管里的电子行为，由量子力学描述。

才有了芯片，有了集成电路，有了你手上这台设备。

而现在，我们正站在第二次量子革命的门槛上。

不再是被动利用量子效应，而是主动操纵叠加和纠缠，去计算、通信、感知。

量子计算机在某些问题上，能碾压所有经典计算机。量子密钥分发已经实用化，中国建成了京沪量子通信干线。

量子雷达、量子成像、量子传感，也在不断提高精度。

量子力学告诉我们，世界在最底层是概率的、关联的、不确定的。

拥抱这种不确定性，也许，能看到更深层的真相。
