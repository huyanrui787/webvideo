# 口播稿

你只要说一句话，代码就写好了。

不是科幻，这东西叫 Codex。

---[hook]

"帮我把表格画成柱状图，红色，标题叫销售趋势。"

话音刚落，Python 代码跳出来了。

图也跑出来了。几秒钟的事。

---[build]

Codex 全名叫 OpenAI Codex，它爹是 GPT-3。

GPT-3 像个博览群书的文科生，什么话题都能聊。

Codex 是丢进代码海洋专门练出来的，理工科学霸。

你把自然语言扔给它，它给你翻译成代码。

支持的语言多到吓人。

Python、JavaScript、Go、Ruby、Perl，它都会。

其中 Python 尤其炉火纯青。

它怎么练出来的？四个字：海量代码。

训练数据来自 GitHub 上千万个公开仓库。

每行代码前面有注释，后面是实现。

这就是最好的"人话→代码"对照教材。

---

光看不够，还得做题。

OpenAI 搞了个评测集叫 HumanEval。

里面 164 道手写编程题。

每道题有人话描述，配上测试用例。

Codex 就在这个循环里反复练。

理解题目、生成代码、跑测试、看哪错了、再修正。

有意思的是，生成答案只要几毫秒。

但验证答案，得真的运行代码。

这倒逼它去理解代码的执行逻辑。

---

训练数据什么都有。

教科书式的优雅代码，也有初学者写的 bug 和屎山。

鱼龙混杂反而让它见多识广。

既学了最佳实践，也认识了各种低级错误。

当然，这也给后来翻车埋了伏笔。

它最出名的产品叫 GitHub Copilot。

你在 VS Code 里写代码，它像个幽灵搭档。

写一行注释，它把整个函数补完。

连错误处理都帮你配齐。

有时候你只是写了个函数名，它就猜出你想干嘛。

---[build]

别以为它只会自动补全。它能做的事多多了。

先说交互式编程。

你写清楚需求：从 API 取五个城市的天气，整成表格，温度高于 30 度的标红。

它几十行代码瞬间生成。

requests 库调好，pandas 整好，matplotlib 画图格式都设好。

你像建筑设计师，它是高效施工队。

---

跨语言翻译也拿手。

公司要把 Java 旧服务迁移到 Go。

你不用自己啃上千行代码。

把 Java 代码喂给 Codex，加一行注释：翻成 Go。

它就给你一个像模像样的 Go 版本。

变量命名、错误处理风格，全是 Go 的地道写法。

---

看不懂别人的天书代码？

贴给 Codex，让它解释。

它会一层层剥开：先从数据库查用户，然后检查权限，权限不足就跳转。

像给每段代码配了个即时讲解员。

---

随手写游戏也行。

你告诉它：做个网页端乒乓球游戏，键盘控制左边板子。

HTML、CSS、JS 全套给你生成。

画面简陋，但真的能玩。

你还能继续改：把球变慢、给背景加上星空。

不会编程的人，第一次尝到做产品的甜头。

---

数据处理更不在话下。

乱糟糟的日志文本，要提取 IP 地址和时间戳。

这种让人头疼的手工活，它眨眼搞定。

写正则的功力，比很多老手都利索。

---[turn]

但是，它翻起车来能吓死人。

---[data]

先说最常见的：一本正经地胡说八道。

AI 圈有个专业词叫幻觉。

你问一个冷门库的用法，它可能没怎么学过。

但为了完成任务，它会凭空编造。

编几个看上去特别合理、实际根本不存在的函数名。

你复制粘贴，程序一跑就崩。

查了半天才发现，那函数是它臆想出来的。

---

更头疼的是安全漏洞。

训练数据里有大量社区代码，质量参差不齐。

里面藏着 SQL 注入、跨站脚本攻击。

Codex 学去了，生成的代码直接继承这些漏洞。

你让它写个登录接口，它流畅写完。

但用户输入被直接拼进 SQL 查询语句。

一个经典的注入漏洞就这么被制造出来了。

就像厨师学会了切菜，但不知道把烂菜叶扔掉。

你吃下去就容易闹肚子。

---

复杂逻辑它也不行。

让它写排序算法，完全没问题。

但你说：设计一个电商折扣系统。

满 200 减 30，会员额外八折，限时特价不参与。

这种条件互斥、需要权衡的业务逻辑。

它的脑子就开始混乱了。

它擅长一步到位、定义清晰的指令。

一旦涉及多步骤、需要取舍的宏观设计，就转不过弯。

说到底，它只是模式匹配大师。

不是真正理解"为什么这样设计"的工程师。

---

还有偏见问题。

它可能输出含有冒犯性变量名的代码。

这些都源于训练数据里藏着的暗影。

---[release]

一个能帮你写代码的 AI，同时也在帮你埋雷。

---[close]

程序员会失业吗？短期内，答案是否定的。

它更像一双超级跑鞋。

以前你自己走路，现在能跑更快、跳更高。

但往哪儿跑，还得你的大脑来指挥。

它会吃掉那些重复的模式化劳动。

样板代码、格式转换、单元测试初稿。

这些大概率逐渐交给 AI。

程序员的核心价值往上移。

需求分析、系统架构、复杂调试、价值判断。

这些才是你的主战场。

---

计算器没让数学家失业。

反而让他们专注于更抽象的理论。

搜索引擎没让学者失业。

反而极大扩展了他们获取信息的边界。

Codex 也一样。

它让写代码这件事变得更民主。

不会编程的产品经理、设计师、科研人员。

靠详细描述需求，也能快速做出原型，验证想法。

---

未来最稀缺的能力是什么？

把复杂模糊的现实问题，拆成 AI 能精准执行的明确指令。

这种表达能力和系统思维，比以前更值钱。

你不再是代码的敲字工。

你是 AI 代码生成器的指挥家。

---

Codex 能完美执行画图的指令。

但它不会在深夜突发奇想，想画一幅关于孤独的星空。

你的灵巧的手，加上会发问、会感受、会做梦的大脑。

才是这个时代永远无法被替代的筹码。

把枯燥和重复留给 AI，把探索和定义留给自己。
