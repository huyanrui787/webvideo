# 口播稿

> 总时长约 65s · 信息保留度 > 60%

你还记得吗？十几年前在网页上看视频，浏览器弹出一个框——需要安装 Flash 插件。

---[hook]

加载半天转圈圈，动不动崩溃。手机端？直接不支持，得跳去 App。

---[hook]

直到 HTML5 来了。一个 `<video>` 标签，跟插图片一样简单，浏览器自己就能播。

---[data]

你写这一行就够了：`<video src="视频.mp4" controls></video>`，播放按钮、进度条、音量，全自动配好。

---[data]

想自动播？加 `autoplay`。想循环？加 `loop`。跟搭积木似的，属性可以随便组合。

---[data]

但你以为这就完了？真正好玩的才刚开始。

---[turn]

默认控件千篇一律？关掉它，自己用 JS 造一个。圆形播放按钮、自定义进度条、倍速、画中画，全在你手里。

---[build]

担心浏览器不兼容？用 `<source>` 标签，mp4、webm、ogg 三种格式全丢进去，浏览器自己挑。

---[build]

还有字幕。加个 `<track>` 标签，加载 vtt 文件，听障用户也能看懂你的视频。

---[build]

一行代码打开视频，几行代码让它完全听你的。

---[release]

打开编辑器，新建个 .html 文件，把 `<video>` 标签敲进去试试。你会发现，用 HTML 做视频，真的比想象中简单一百倍。

---[close]
