# 节奏蓝图

> 总时长：约 65s · 情绪弧：方法论型（痛点 → 旧方法失败 → 新方法揭示 → 演示 → 呼吁）

## 节拍规划

| # | 类型 | 时长 | 情绪目标 |
|---|------|------|---------|
| 1 | [hook] | ~10s | 痛点共鸣：唤起 Flash 时代看视频的集体痛苦记忆 |
| 2 | [data] | ~12s | 信息冲击：HTML5 一个 `<video>` 标签就搞定，属性如搭积木 |
| 3 | [turn] | ~10s | 颠覆预期：不只简单——还能自定义播放器、字幕、画中画 |
| 4 | [build] | ~16s | 能力堆叠：格式兼容、JavaScript API、track 字幕一网打尽 |
| 5 | [release] | ~6s | 情绪释放：一行代码，浏览器就是最好的播放器 |
| 6 | [close] | ~8s | 行动号召：打开编辑器，试试你的第一个 video 标签 |

## 自检

- [x] [turn] 落在全文 34% 位置（~22s / 65s）
- [x] [release] 在信息最密集的 [build] 之后
- [x] [hook] ≤ 15s，[release] ≤ 8s
- [x] 总时长之和 ≈ 65s（误差 < 10%）
- [x] 情绪弧从模板库选择：方法论型
